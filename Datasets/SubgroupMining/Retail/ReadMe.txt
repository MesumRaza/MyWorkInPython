Structure:

The following dataset was donated by Tom Brijs and contains the (anonymized) 
retail market basket data from an anonymous Belgian retail store.

Each transaction (row) contains the ids of the purchased items.

There are 88162 transactions (rows) in total.
16469 distinct items.
